## 问题与决策记录

### 已决策

| 日期 | 问题 | 决策 | 原因 |
|------|------|------|------|
| 2026-03-02 | 是否使用并行多实例 Claude Code | ❌ 不采用，使用单实例 + 顺序任务 | Unity 编译是串行瓶颈，并行收益为零；stdio transport 下各进程 OperationQueue 互不共享 |
| 2026-03-02 | MCP Server 传输协议 | ✅ stdio（StdioServerTransport） | 无需长驻进程，按需启动，架构更简单 |
