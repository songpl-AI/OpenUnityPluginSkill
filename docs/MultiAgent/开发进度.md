## 问题与决策记录

### 已决策

| 日期 | 问题 | 决策 | 原因 |
|------|------|------|------|
| 2026-03-02 | 是否使用并行多实例 Claude Code | ❌ 不采用，使用单实例 + 顺序任务 | Unity 编译是串行瓶颈，并行收益为零；stdio transport 下各进程 OperationQueue 互不共享 |
| 2026-03-02 | MCP Server 传输协议 | ✅ stdio（StdioServerTransport） | 无需长驻进程，按需启动，架构更简单 |
| 2026-03-02 | MCP Server 工具参数 schema | ✅ zod（`@modelcontextprotocol/sdk` 原生支持） | 类型安全，比 JSON Schema 更简洁；空参数工具传 `inputSchema: {}` |
| 2026-03-02 | 嵌套 GameObject 路径的 URL 编码 | ✅ `encodeURIComponent(path)` | 路由器 `[^/]+` 不匹配斜杠；C# 端 `Uri.UnescapeDataString` 自动解码 |
| 2026-03-02 | 新增组件相关工具（原缺失导致脚本无法挂载） | ✅ 补充 3 个工具 | C# 后端已有 HandleAddComponent，TypeScript 侧之前未暴露 |
| 2026-03-02 | MCP Server TypeScript 模块系统 | ✅ ESM（`"type": "module"`, `"module": "Node16"`） | MCP SDK 使用 `.js` 扩展名导入，与 CommonJS 不兼容 |

### 待决策

| 问题 | 选项 | 所需信息 |
|------|------|---------|
| `--allowedTools "mcp__unity-editor__*"` 通配符是否支持 | 实测确认 | 执行 1.11 时验证，不支持则改为逐一列出 |
