## 已知风险 & 阻塞项

| 风险 | 影响 | 缓解方案 |
|------|------|----------|
| OpenClaw Plugin SDK 文档不完整 | 影响阶段三实现 | 参考社区插件源码反向工程 |
| Unity 主线程调度性能 | 高频调用时响应慢 | 批量操作 API 设计 |
| AssemblyReload 后端口占用 | Server 无法重启 | 增加端口释放重试逻辑 |
| websocket-sharp 与 Unity Mono 兼容性 | 影响 2021.3 WebSocket 稳定性 | 充分测试 + 备选方案（降级为轮询） |
| 脚本反射在编译失败时不可用 | 项目理解能力降级 | 返回文件列表作为降级响应，并在 SKILL.md 中说明 |
| 大型项目引用反查耗时 | 首次调用体验差 | 增量缓存 + 异步返回 + 超时提示 |
| ClawHub 发布流程未知 | 影响阶段六 | 提前研究发布文档 |
